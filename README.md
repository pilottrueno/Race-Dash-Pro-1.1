Race Dash Pro

Что такое Race Dash Pro

Race Dash Pro — это профессиональная цифровая приборная панель для спортивных и тюнингованных автомобилей, построенная на Raspberry Pi.

Проект объединяет современный интерфейс, высокую скорость работы, поддержку различных ECU, расширенную телеметрию, запись логов, GPS-функции и физическую светодиодную индикацию переключения передач.

Основная цель проекта — создать полностью самостоятельный продукт с собственным фирменным стилем, не копируя существующие решения.

Адаптивный тахометр

Тахометр автоматически меняет цвет в зависимости от оборотов двигателя.

Например:

1000 RPM — голубой

3000 RPM — зелёный

5000 RPM — жёлтый

6500 RPM — оранжевый

7500 RPM — красный

8000 RPM — пульсация красным

Все переходы выполняются плавно.

Race Dash Pro использует полностью анимированный интерфейс.

Поддерживаются:

плавное изменение показаний;

динамические градиенты;

Boot Animation

При включении зажигания отображается фирменная загрузка.

Этапы:

появление логотипа Race Dash Pro;

подпись Designed & Engineered by Bukhlakoff Motorsport;

проверка системы;

подключение ECU;

подключение GPS;

тест тахометра;

тест физических светодиодов Shift Light;

плавный переход к главному экрану.

Shift Light

Поддерживается внешняя светодиодная линейка SK6812.

Возможности:

настройка количества светодиодов;

настройка цветов;

настройка оборотов включения;

индивидуальные режимы;

мигание на отсечке;

режим Launch Control;

режим Anti-Lag;

режим диагностики;

анимация самотестирования.

Поддержка ECU

Планируется поддержка:

MegaSquirt

Speeduino

Link ECU

ECUMaster

CAN Bus

OBD-II

Демо режим

При отсутствии ECU автоматически включается Demo Mode.

В нём:

моделируется работа двигателя;

изменяются обороты;

рассчитывается BOOST;

изменяются температуры;

работают все анимации;

работают графики;

работают предупреждения.

Это позволяет тестировать приборную панель без автомобиля.

Графики

Для каждого датчика доступны графики.

Можно отображать:

относительно времени;

относительно оборотов двигателя.

Поддерживается масштабирование, прокрутка и выбор нескольких параметров.

Логирование

Race Dash Pro автоматически записывает данные.

Поддерживается:

логирование каждую секунду;

автоматическое создание минутных файлов;

просмотр логов;

экспорт TXT;

экспорт CSV;

экспорт для анализа в специализированных программах.

Предупреждения

Система предупреждает водителя при выходе параметров за безопасные пределы.

Например:

низкое давление масла;

перегрев масла;

перегрев охлаждающей жидкости;

бедная смесь;

высокая температура выхлопных газов;

низкое напряжение аккумулятора.

Предупреждения отображаются цветом, анимацией и при необходимости могут сопровождаться внешней световой индикацией.

Философия проекта

Race Dash Pro создаётся как независимая цифровая приборная панель с собственным узнаваемым стилем. Главные принципы — высокая скорость работы, наглядность, удобство для водителя и возможность глубокой настройки под разные автомобили и разные сценарии использования: от повседневной езды до автоспорта.
